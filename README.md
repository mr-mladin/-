# VK Ads Sync

Автосинк VK Ads -> Google Sheets.

## Income Tracker (MVP)

Лёгкая веб-страничка для учёта доходов: статичный HTML/CSS/JS, без бэкенда. Данные хранятся в `localStorage` браузера.

**Запуск:**

```bash
# вариант 1 — просто открыть в браузере
open index.html      # macOS
xdg-open index.html  # Linux

# вариант 2 — локальный сервер (если браузер блокирует localStorage по file://)
python3 -m http.server 8000
# затем http://localhost:8000
```

**Возможности:**

- добавление дохода (сумма, дата, категория, комментарий)
- сводка: общий доход, количество записей, средний чек, топ-категория
- фильтры по периоду (месяц / год / всё время) и поиск
- экспорт в CSV
- адаптивный светлый дизайн
