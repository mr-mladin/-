# Финансы

Веб-приложение для учёта личных финансов. Облачная синхронизация через Supabase, авторизация по email/паролю, работает на десктопе и мобильном.

## Возможности

- Счета (карты, наличные, накопления) — собственный порядок, валюта, цвет, архив
- Операции: доход, расход, перевод между счетами; редактирование, дублирование, удаление
- Категории доходов и расходов с подкатегориями (3 уровня), теги
- Дашборд: доход/расход/остаток за выбранный финансовый месяц, остатки по счетам, топ категорий
- Полнотекстовый поиск и фильтры по счетам, категориям, тегам, датам и суммам
- Бюджеты по категориям расходов на месяц (с подсчётом перерасхода)
- Цели накоплений с прогрессом
- Темы: светлая, тёмная, авто
- Настройки: основная валюта, формат сумм, первый день недели, начало финансового месяца
- Экспорт и импорт CSV (банковские выписки)

## Стек

- **Preact + htm** — UI без сборки, импорты прямо в браузере через ESM CDN
- **Supabase** — PostgreSQL + Auth (бесплатный тариф)
- **GitHub Pages** — деплой статики

## Локально

```bash
python3 -m http.server 8000
# затем открыть http://localhost:8000
```

## VK Ads Sync (старое)

В этом репозитории также живёт `sync_vk_ads_to_sheets.py` — отдельный скрипт автосинка VK Ads → Google Sheets, не связанный с веб-приложением.

### VK Ads Balance Sync

`sync_vk_ads_balances_to_sheets.py` обновляет баланс кабинетов VK Ads в основном месячном листе Google Sheets.

- Workflow: `.github/workflows/vk-ads-balances.yml`
- Расписание: каждые 30 минут
- Для пилота баланс пишется в колонку `A` на 8 строк ниже названия проекта:
  - Оксана: `A10`
  - Ксения: `A27`

Нужный GitHub Secret:

- `VK_ADS_BALANCE_CLIENTS_JSON` — JSON со списком кабинетов и их `client_id` / `client_secret`

Полезные GitHub Variables:

- `VK_ADS_BALANCE_ACTIVE_ONLY=false`
- `VK_ADS_TIMEZONE=Europe/Moscow`
- `VK_ADS_MAPPING_SHEET=Сопоставление`
